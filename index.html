How It Works

Three steps. Full coverage.

No SDK wrapping, no agent modifications, no middleware injection. Install once, monitor everything.

1

Install

One pip install drops the policy agent, classifier, and persistent hook onto your system.

pip install breach-intel-client
2

Hook

The install-hook writes a sitecustomize.py that auto-patches every Python agent at import time.

breach-intel install-hook
3

Run

Start your agents normally. Every LLM response is classified in <1ms. Breaches log to DB, JSONL, and webhooks.

python my_agent.py
