Frequently asked questions

Can I try Breach Intel for free?

Yes. The Community plan is free forever and available on PyPI right now. Install with pip install breach-intel and start logging breaches immediately.

Do I need to modify my agent code?

No. Breach Intel uses a persistent Python hook (sitecustomize.py) to auto-patch supported frameworks at import time. Your agent code stays completely untouched.

What happens if the policy agent goes down?

The SDK has a 5-second timeout and fails silently. Your agent continues running normally — breach logging resumes when the policy agent recovers. No agent downtime, ever.

Can I deploy on-premise?

Yes, with the Enterprise plan. Breach Intel ships as a Docker container and can run in your VPC, air-gapped environments, or any infrastructure with Docker/K8s support.
