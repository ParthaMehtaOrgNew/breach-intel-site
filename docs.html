Get started with Breach Intel

Everything you need to install, configure, and run breach detection on your AI agents.

Quick Start

Up and running in 3 commands

# 1. Install the SDK $ pip install breach-intel-client # 2. Install the persistent hook $ breach-intel install-hook # 3. Set your policy agent URL and run $ export BREACH_INTEL_URL=http://localhost:8080 $ python my_agent.py ✓ Agent auto-registered. Breach detection active.
